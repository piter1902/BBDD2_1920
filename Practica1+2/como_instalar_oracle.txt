Para instalar Oracle hay que descargar:

https://oss.oracle.com/debian/dists/unstable/non-free/binary-i386/oracle-xe_10.2.0.1-1.1_i386.deb

Instalaremos las dependencias con:

apt-get install libaio1
apt-get upgrade libaio
apt-get upgrade glibc

Instalaremos el paquete con:

apt-get install bc
dpkg --install oracle-xe_10.2.0.1-1.1_i386.deb
/etc/init.d/oracle-xe configure

Instalaremos el SGBD
